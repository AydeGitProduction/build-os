## Blockers Resolved (Chronological)

### 1. GitHub App PKCS#1 / OpenSSL 3.x incompatibility
- **Root cause:** Node 18+ / OpenSSL 3 dropped PKCS#1 PEM loading from default provider
- **Fix:** `loadPrivateKey()` ‚Äî try `createPrivateKey(pem)`, fall back to `{ format:'der', type:'pkcs1' }` DER load

### 2. Base64-encoded PEM in Vercel env
- **Root cause:** `GITHUB_APP_PRIVATE_KEY` was stored as `base64(pemFile)` ‚Äî raw first bytes were `LS0tLS1CRUdJTiBSU0Eg` = base64("-----BEGIN RSA ")
- **Fix:** `normalizePrivateKeyPem()` ‚Äî step 3 detects and decodes base64-encoded PEM before wrapping

### 3. GitHub App lacks Administration: Read & Write permission
- **Root cause:** Installation token (ID 119933236) returned `403 Resource not accessible by integration` on `POST /user/repos`
- **Fix:** Created PAT `buildos-provision-v2` (`[REDACTED_PAT]`) with `repo,delete_repo` scope; PAT takes priority in `resolveGitHubToken()`

### 4. `AydeGitProduction` is a user account, not an org
- **Root cause:** `POST /orgs/AydeGitProduction/repos` ‚Üí 404
- **Fix:** Probe `GET /orgs/{owner}` ‚Äî if 404, use `POST /user/repos`

### 5. Vercel "Login Connection" missing
- **Root cause:** Vercel account `admin-28708031` had no GitHub identity linked
- **Fix:** Authorized Vercel OAuth app on `AydeGitProduction` via `/account/settings/authentication`

### 6. Vercel GitHub App not installed
- **Root cause:** `POST /v10/projects` with `gitRepository` returned `400 ‚Äî GitHub integration not installed`
- **Fix:** Installed Vercel GitHub App on `AydeGitProduction` via `github.com/apps/vercel/installations/new` (installationId: 120238047)
